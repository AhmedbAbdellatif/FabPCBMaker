# FabPCBMaker
A fabbable, cheap, light and small CNC for PCBs

## Project Development
I developed this project as a final project for FabAcademy 2017. My idea was to make it possible to make a CNC mill for PCBs in a FabLab at an as low as possible cost (less than 100â‚¬). 
In this page I will be providing Information on how to replicate this along with the design files required

## Specifications
* 100x160x50mm working area
* 0.0025mm theorethical resolution (0.04mm realistic)
* 18mm plywood cnc milled frame
* POM bearing and motor holder
* brushless spindle motor
* 3 x Pololu A4988
* 3 x Nema 17 threaded rod stepper motors
* 8mm linear rods
* satshakit board
